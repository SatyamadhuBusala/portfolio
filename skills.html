DOCKER

Docker is an open-source platform that enables developers to build, package, and run applications in lightweight containers. Unlike traditional virtual machines, containers share the same OS kernel but are isolated from each other, making them faster, portable, and efficient.

JENKINS

Jenkins is an open-source automation tool used for building, testing, and deploying software. It helps developers automate the entire software development process (CI/CD) by running tasks whenever code is changed. With Jenkins, teams can deliver software faster and with fewer errors.

KUBERNETES

Kubernetes (K8s) is an open-source platform that automates the deployment, scaling, and management of containerized applications. It helps run containers across multiple machines, making applications more reliable, scalable, and easy to manage.

MYSQL

MySQL is an open-source relational database management system (RDBMS) that stores and organizes data in tables. It uses Structured Query Language (SQL) to manage, retrieve, and update data, making it widely used for web applications and enterprise systems.

TERRAFORM

Terraform is an Infrastructure as Code tool that allows you to define and manage cloud resources using code. It helps automate the setup of servers, networks, and other infrastructure across multiple cloud providers in a consistent and repeatable way.
